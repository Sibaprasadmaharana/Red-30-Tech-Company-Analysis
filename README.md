# Red-30-Tech-Company-Analysis
This project includes SQL queries for the Red30Tech company's database. The queries cover a range of tasks, including analyzing online retail sales, extracting session information, finding high-order total orders, and implementing various window functions.
Tasks Covered:

Identify orders with a total higher than the average total price.
Retrieve session information for Two Trees Olive Oil employees.
Extract information about conference attendees from states with no online retail sales.
Find inventory items with quantities less than the average.
Get each customer's most recent order using ROW_NUMBER().
Retrieve the top 3 orders with the highest total from each product category purchased by Boehm Inc.
Use LAG() and LEAD() to show the previous and next Red30 Tech sessions in Room 102.
Use LAG() to track the quantity of Drones ordered on the previous 5 order dates.
Use RANK() and DENSE_RANK() to rank employees alphabetically by last name.
Pull registration information for the first three people that registered in each state using DENSE_RANK().
Instructions
Setting Up the Database:

Ensure you have access to the Red30Tech database.
Execute the provided SQL queries in a SQL Server environment to create the necessary tables.
Populate the tables with sample data.
Running the Queries:

Each query is well-documented and labeled with a task number.
Copy and paste the queries into your SQL Server Management Studio or preferred SQL tool.
Execute the queries to see the results.
